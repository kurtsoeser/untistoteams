MS365-Schulverwaltung

Werkzeug: SharePoint – Lehrerliste

Lehrkräfte aus den Schul-Grundeinstellungen (Stammdaten) als neue SharePoint-Liste auf einer von Ihnen gewählten Website anlegen – per Microsoft Graph.

Hilfe

Dashboard
Berechtigungen: Sites.ReadWrite.All (delegiert, Admin-Zustimmung). Die angemeldete Person braucht Schreibrechte auf der Ziel-Website.

Ziel

Es wird eine generische Liste mit Spalten Titel (Name), Kürzel, E-Mail, UPN (vorerst gleich E-Mail) erzeugt und mit allen Lehrkräften aus den Einstellungen befüllt. Bereits vorhandene gleichnamige Liste → Fehler von Graph.
