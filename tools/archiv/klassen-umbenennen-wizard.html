Archiv: Vier‑Schritt‑Assistent. Der empfohlene Weg ist jetzt die Schul‑Einstellungen → Tab „Klassen“ (Block „Klassen‑Gruppen … Anzeigename nach Schuljahreswechsel“).

Schritt 2: Umbenennungsregel

Filter (Format-Kriterium): Umbenannt werden nur Gruppen, deren Anzeigename dem Muster entspricht, z. B. Klasse 1A, Klasse 2B oder Klasse 10A: fester Text Präfix, dann die Stufe (Zahl), dann optional ein Kürzel (z. B. A, B, HAK).
Schuljahreswechsel: Die Stufe wird automatisch um +1 erhöht (z. B. Klasse 1A → Klasse 2A, Klasse 2B → Klasse 3B).

Bulk: Text ersetzen (optional)

Beispiel: Klasse → Jahrgang (wird auf den neuen Anzeigenamen angewendet).

Schritt 4: Umbenennung ausführen

Achtung: Es werden die in der Vorschau gültigen Gruppen per PATCH /groups umbenannt. Bitte zuerst die Vorschau prüfen. Der Vorgang kann bei vielen Gruppen einige Minuten dauern. Dieses Tool ändert dabei nur den Anzeigenamen (displayName). Mail-Nickname und Gruppen-E-Mail bleiben unverändert.
