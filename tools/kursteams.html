Schritt 8 (optional): Schüler als Mitglieder hinzufügen (PowerShell + Microsoft Graph)

Dieser Schritt ist optional. Er erzeugt ein PowerShell‑Script (Microsoft.Graph), das nach der Team‑Anlage Schüler als Mitglieder in die jeweiligen Gruppen/Teams einträgt.
Format: pro Zeile Klasse;UPN oder Klasse;Gruppe;UPN (Gruppe = z. B. G1/G2 für Sprachgruppen).
Wenn ein Kursteam eine Gruppe (z. B. G1) hat, wird bevorzugt die passende Klasse;Gruppe‑Liste verwendet, sonst die Klassenliste.

Microsoft Graph im Browser

⚠️ Wichtig: Führen Sie das Script im selben Verzeichnis wie die neueteams.csv aus. Admin-Anmeldung erforderlich!

PowerShell-Script:

$TeamsList = Import-Csv -Path .\neueteams.csv -Encoding UTF8
Connect-MicrosoftTeams

$i = 0
foreach ($Team in $TeamsList) {
    $i++
    try {
        $null = New-Team -Template "EDU_Class" -DisplayName $Team.TeamName -MailNickName $Team.Gruppenmail -Owner $Team.Besitzer -ErrorAction Stop
        Write-Host "OK [$i/$($TeamsList.Count)] $($Team.Gruppenmail)"
    }
    catch {
        Write-Warning "Fehler [$i] $($Team.Gruppenmail): $($_.Exception.Message)"
    }
    Start-Sleep -Seconds 2
}
