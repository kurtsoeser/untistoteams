Freigegebene Postfächer

Exchange Online Export (SharedMailbox → JSON)
1) Script kopieren oder als .ps1 speichern und in PowerShell ausführen.
2) Es wird eine Datei shared-mailboxes-export.json erzeugt.
3) Diese Datei hier mit Import (JSON) laden.

    Details

    Links ein freigegebenes Postfach auswählen, Neu für ein Erstellungs-Script öffnen oder Tenant einlesen starten.
    Neues freigegebenes Postfach: Fülle die Stammdaten und optional Delegierte aus. Das Script erstellt das Postfach mit Exchange Online PowerShell und setzt danach die gewählten Rechte.
    Ohne Leerzeichen/Sonderzeichen, z. B. der lokale Teil der Adresse.
    Optional, aber empfohlen. Ohne Angabe verwendet Exchange die Standarddomain des Tenants.
    PowerShell: Für Berechtigungen (FullAccess/SendAs/SendOnBehalf) ist Exchange Online maßgeblich. Das Script unten liest Berechtigungen aus – und kann optional Änderungen setzen.
