Werkzeug: SharePoint – Websiteerstellung & Speicher

Mandantenweite Einstellungen aus dem SharePoint Admin Center: wer Sites anlegen darf, Pfade, Standardzeitzone und Speicherlimit für neue Sites – per Microsoft Graph (GET/PATCH /admin/sharepoint/settings).

Hilfe

Dashboard
Berechtigungen: SharePointTenantSettings.Read.All (Lesen) bzw. SharePointTenantSettings.ReadWrite.All (Speichern) – mit Administratorzustimmung. Der angemeldete Benutzer muss laut Microsoft SharePoint Administrator oder Global Administrator sein.

Werte

Wenn aktiviert, kann ein festes Kontingent pro neuer Site sinnlos sein – im Admin Center prüfen.
